Zombie Knight
- A 2D action game developed using Python and Pygame.


How to Run
- Download the ZombieKnight.7z file.
- Extract the archive using 7-Zip.(To Download 7-zip go to https://www.7-zip.org/)
- Open the extracted folder.
- Run main.exe.


Features
- 2D action gameplay
- Animated player and enemies
- Sound effects and music
- Portal system and level progression
